Parameter balancing can be configured via several options. The options are
  • use_pseudos (True or False): Since independent priors can capture only some types of kinetic constants (the mutually independent ones), pseudo values can be used, in addition, to represent prior knowledge on other, dependent kinetic constants (default: True).
  • pH (float): pH value that is assumed as modelling condition (default: 7).
  • temperature (float; in Kelvin): temperature value that is assumed as modelling condition (default: 300).
  • overwrite_kinetics (True or False): The kinetic laws of the SBML model can either be kept or overwritten with convenience kinetics and inserted balanced parameters (default: True).
  • cell_volume (float; in ÂµM): Volume of the cell the model is part of (default: 43 (yeast)).
  • parametrisation (string): For the convenience kinetics that are inserted in the model, three different parametrisations can be chosen ('hal': Haldane parametrisation; 'weg': Wegscheider parametrisation; 'cat': ;). Please see http://bioinformatics.oxfordjournals.org/content/26/12/1528.full for more details on the parametrisation (default: 'hal'.
  • enzyme_prefac (True or False): Usage of a prefactor for enzyme concentrations (default: True).
  • default_inhibition ('complete_inh','comp', or 'non-comp'): Default type of reaction inhibition, which can be either complete, competetitive or non-competetive (default: 'complete_inh').
  • default_activation ('complete_act','essential', or 'non-essential'): Default type of reaction activation, which can be either complete, essential, or non-essential (default: 'complete_act').
  • model_name (string): A name for the output model and SBtab file (default: inputfilename+'_balanced').
  • samples (int): How many sample models shall be drawn from the posterior distribution?
  • boundary_values ('enforce', 'ignore', or 'warning'): Usage of numerical boundaries: the balanced parameters can either be forced to be within the given boundaries, or they only produce a warning in the log file if they are outside the boundaries.
